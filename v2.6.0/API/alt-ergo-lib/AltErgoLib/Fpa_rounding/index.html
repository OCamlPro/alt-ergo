Module AltErgoLib.Fpa_rounding

module DE = Dolmen.Std.Expr

The rounding modes for the Floating Point Arithmetic theory. In the legacy frontend, the rounding mode type was `fpa_rounding_mode` and defined 5 rounding modes (see the rounding_mode type below). The SMT2 standard defines the exact same rounding modes, but with different identifiers.

type rounding_mode =
  1. | NearestTiesToEven
  2. | ToZero
  3. | Up
  4. | Down
  5. | NearestTiesToAway
